# Dijkstra's Algorithm

This program implements Dijkstra's algorithm for finding the shortest path in a graph in C++.

## Description
Dijkstra's algorithm finds the shortest path from a source vertex to all other vertices in a weighted graph with non-negative edge weights.

## Usage
- Input: Graph (adjacency matrix or list), source vertex
- Output: Shortest path distances from the source

## Time Complexity
- O(V^2) or O(E + V log V) with min-heap
