# Activity Selection Problem

This program solves the activity selection problem using a greedy approach in C++.

## Description
Given a set of activities with start and finish times, select the maximum number of activities that can be performed by a single person, assuming that a person can only work on a single activity at a time.

## Usage
- Input: Arrays of start and finish times
- Output: Maximum set of non-overlapping activities

## Time Complexity
- O(n log n)
