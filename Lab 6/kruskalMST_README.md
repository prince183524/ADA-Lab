# Kruskal's Minimum Spanning Tree (MST)

This program implements Kruskal's algorithm for finding the Minimum Spanning Tree in a graph in C++.

## Description
Kruskal's algorithm finds a subset of the edges that forms a tree including every vertex, where the total weight of all the edges in the tree is minimized.

## Usage
- Input: Weighted undirected graph (edge list)
- Output: Edges in the MST and their total weight

## Time Complexity
- O(E log E)
