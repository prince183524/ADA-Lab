# lab6

This folder contains C++ programs for greedy algorithms and graph algorithms.

## Files
- **activityselection.cpp**: Solves the activity selection problem using a greedy approach.
- **dijkstra.cpp**: Implements Dijkstra's algorithm for shortest paths in a graph.
- **kruskalMST.cpp**: Implements Kruskal's algorithm for finding the Minimum Spanning Tree.
- **prims.cpp**: Implements Prim's algorithm for finding the Minimum Spanning Tree.

Each file contains a self-contained C++ program with input/output via the console.
