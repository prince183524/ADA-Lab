# Max-Min (Divide and Conquer)

This program finds the maximum and minimum values in an array using the divide and conquer approach in C++.

## Description
The algorithm recursively divides the array into subarrays, finds the max and min in each, and combines the results.

## Usage
- Input: Array of integers
- Output: Maximum and minimum values

## Time Complexity
- O(n)
