# Convex Hull

This program finds the convex hull of a set of points in C++.

## Description
The convex hull is the smallest convex polygon that contains all the points. Common algorithms include Graham's scan and Jarvis march.

## Usage
- Input: Set of points (x, y)
- Output: Points forming the convex hull

## Time Complexity
- O(n log n) (Graham's scan)
