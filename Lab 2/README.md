# lab2

This folder contains C++ programs for searching and sorting algorithms.

## Files
- **Binarysearch.cpp**: Implements the Binary Search algorithm.
- **insertionsort.cpp**: Implements the Insertion Sort algorithm.

### lab3 (subfolder)
- **mergesrt.cpp**: Implements the Merge Sort algorithm.
- **quicksort.cpp**: Implements the Quick Sort algorithm.

Each file contains a self-contained C++ program with input/output via the console.
