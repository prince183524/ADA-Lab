# Selection Sort

This program implements the Selection Sort algorithm in C++.

## Description
Selection Sort divides the input list into two parts: the sublist of items already sorted and the sublist of items remaining to be sorted. It repeatedly selects the smallest (or largest) element from the unsorted sublist and moves it to the sorted sublist.

## Usage
- Input: Array of integers
- Output: Sorted array in ascending order

## Time Complexity
- Best: O(n^2)
- Average: O(n^2)
- Worst: O(n^2)
