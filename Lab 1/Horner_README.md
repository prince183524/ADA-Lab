# Horner's Method

This program evaluates polynomials using Horner's method in C++.

## Description
Horner's method is an efficient algorithm to evaluate polynomials at a given value of x, reducing the number of multiplications required.

## Usage
- Input: Coefficients of the polynomial and the value of x
- Output: Value of the polynomial at x

## Time Complexity
- O(n)
