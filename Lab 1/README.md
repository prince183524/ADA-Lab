# lab1

This folder contains C++ programs for basic algorithms and array operations.

## Files
- **Bubble_sort.cpp**: Implements the Bubble Sort algorithm.
- **Duplicateno.cpp**: Finds duplicate numbers in an array.
- **Horner.cpp**: Evaluates polynomials using Horner's method.
- **Selection_sort.cpp**: Implements the Selection Sort algorithm.

Each file contains a self-contained C++ program with input/output via the console.
